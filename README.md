# Calculadora Completa

Uma calculadora interativa com diversas funcionalidades matemáticas e de conversão, projetada para facilitar seus cálculos. Ela oferece operações básicas, funções trigonométricas, estatísticas, regra de 3, Bhaskara, e muito mais.

## Funcionalidades

A calculadora oferece as seguintes opções:

- **[0]** - **Sair**: Encerra a calculadora.
- **[1]** - **Soma**: Realiza a operação de soma entre dois ou mais números.
- **[2]** - **Subtração**: Realiza a operação de subtração entre dois números.
- **[3]** - **Multiplicação**: Realiza a operação de multiplicação entre dois números.
- **[4]** - **Divisão**: Realiza a operação de divisão entre dois números.
- **[5]** - **Conversão de Valores**: Permite converter entre diferentes unidades de medida, como distância, tempo, massa, etc.
- **[6]** - **Bhaskara**: Resolve equações quadráticas utilizando a fórmula de Bhaskara.
- **[7]** - **Porcentagem**: Calcula porcentagens de um valor.
- **[8]** - **Regra de 3**: Resolve problemas envolvendo proporções.
- **[9]** - **Trigonometria**: Calcula funções trigonométricas, como seno, cosseno e tangente.
- **[10]** - **Estatística**: Realiza cálculos estatísticos básicos, como média, desvio padrão e variância.

## Como usar

1. **Iniciar a calculadora**: Ao iniciar o programa, você verá um menu com as opções acima.
2. **Escolher uma operação**: Digite o número correspondente à operação que deseja realizar.
3. **Seguir as instruções**: Dependendo da opção escolhida, a calculadora solicitará entradas adicionais. Por exemplo:
   - Para **soma**, **subtração**, **multiplicação** ou **divisão**, você precisará inserir dois números.
   - Para a opção de **Bhaskara**, você deve fornecer os coeficientes da equação quadrática.
   - Para a **regra de 3**, a calculadora pedirá os valores para montar a proporção.

## Exemplo de uso

- **[1] - Soma**: Ao escolher a opção **[1]**, a calculadora pedirá que insira dois números. Após isso, ela realizará a soma e exibirá o resultado.

- **[9] - Trigonometria**: Ao escolher a opção **[9]**, será solicitado que insira o valor de um cateto e da hipotenusa, dependendo da escolha, para calcular o **seno**, **cosseno** ou **tangente**.
